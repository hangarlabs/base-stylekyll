Theme

_theme.scss: reusable components and base styles that can be used across the site. You can find the import of the custom variables and bootsrap components.

Variables

_variables.scss: in this file you can find all the variables used for building the theme. Also is the place where you can override bootstrap variables such as the grid and most of the components (modals, dropdowns) etc..

How to override bootstrap variables ?

Find the variable in bootstrap's _variables and then copy the variable you want to override and remove the !default; text and of course update it with the new value.

Twitter Bootstrap

_bootstrap.scss: we included all the components and helpers we need from twitter bootstrap.
