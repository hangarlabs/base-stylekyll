UI Elements

The UI Elements are the basic building blocks of a website.

4.1 Buttons - buttons.scss

A majority of buttons in the site are built from the same twitter bootstrap class.

  • .btn-default -

    Indicate that the button is the default feature of the page.

  • .btn-primary -

    Indicate that the button is the primary feature of the page.

  • .btn-secondary -

    Indicate that the button is the secondary feature of the page.

Link
.btn-default Link
.btn-primary Link
.btn-secondary Link 
<a href="#" class="btn">Link</a>
<button class="btn {$modifiers}">Button Element</button>
